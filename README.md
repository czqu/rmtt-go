# Remote Message  Telemetry Transport(RMTT) Go Client
This repository provides an implementation of the RMTT (Remote Message Telemetry Transport) protocol in Go. 

## Overview

The RMTT Go client implementation, primarily inspired by the popular MQTT Go client, [paho.mqtt.golang](https://github.com/eclipse/paho.mqtt.golang). It offers a similar API structure and design, making it familiar and easy to use for developers experienced with MQTT.

## Installation

To install the RMTT Go client, you can use `go get`:

```bash
go get -u github.com/czqu/rmtt-go
